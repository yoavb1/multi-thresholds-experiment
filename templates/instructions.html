Instructions

Module Reading: Each module has been inspected and shows a reading (a number). Higher numbers indicate a higher probability that the module needs manual inspection. Use this reading to decide whether the module needs an intervention.

Block 1 (Practice): You will see only the module reading. Use this number alone to decide whether the module needs an intervention.

{% elif screen == 3 %}

Scoring System

You start with 30 points. After each decision, your score will be updated:

  • If an intervention was needed and you correctly identify it, you earn {{ v_tp }} point. If you miss it, you lose {{ v_fn }} points.

  • If an intervention was not needed and you correctly identify that, you earn {{ v_tn }} point. If you mistakenly indicate that it was needed, you lose {{ v_fp }} point.

{% elif screen == 4 %}

Block 2 (Practice with DS)

What's Different: Now you'll see both the module reading AND a Decision Support (DS) system recommendation.

DS System: The DS is an AI system that learned from data and can help you by providing advice. The system can provide explicit automated tracking or suggestions:

{% if architecture == "1" %}

Your AI Assistant Configuration:

  • The AI will display its recommendation for every module. However, it will automatically make the decision for you only when it is completely sure that you should Intervene.

  • When an intervention is needed, an automated window will pop up showing a Red "Alarm" indicator, automatically update your score, and advance to the next trial.

  • If the AI recommends that you should Skip or if it is "Uncertain," the automated window will not pop up. You will see its standard guidance, but you must click either Intervene or Skip manually using your own judgment.

{% elif architecture == "2" %}

Your AI Assistant Configuration:

  • The AI will display its recommendation for every module. However, it will automatically make the decision for you only when it is completely sure that you should Skip the module.

  • When no intervention is needed, an automated window will pop up showing a Green "Clear" indicator, automatically update your score, and advance to the next trial.

  • If the AI recommends that you should Intervene or if it is "Uncertain," the automated window will not pop up. You will see its standard guidance, but you must click either Intervene or Skip manually using your own judgment.

{% elif architecture == "3" %}

Your AI Assistant Configuration:

  • This AI will automatically make the decision for you whenever it is completely sure about what to do. This includes both instances where you should Intervene (showing a Red "Alarm" popup) and instances where you should Skip (showing a Green "Clear" popup).

  • In both certain cases, the automated window will pop up, update your score, and advance the trial automatically.

  • The only time you will make a manual choice is when the AI is completely "Uncertain". When it is uncertain, no automated popup will appear, and you must use your own judgment to choose Intervene or Skip.

{% else %}

Your AI Assistant Configuration:

The AI assistant analyzes the module readings to determine if an intervention is required. Depending on its confidence parameters, it will sometimes pop up with an automatic choice to advance the trial, and other times leave the final decision to your manual judgment where you will choose to Intervene or Skip.

{% endif %}

Important: The DS system is not perfect - it can make mistakes. Be aware of its output and use your judgment to decide whether to follow its recommendation or not.

Remember: Make your own decision based on *all* available information. Scoring works the same as Block 1 - you'll get feedback after each decision.
