Instructions

Module Reading: Each module has been inspected and shows a reading (a number). Higher numbers indicate a higher probability that the module needs manual inspection. Use this reading to decide whether the module needs an intervention.

Block 1 (Practice): You will see only the module reading. Use this number alone to decide whether the module needs an intervention.

{% elif screen == 3 %}

Scoring System

You start with 30 points. After each decision, your score will be updated:

  • If an intervention was needed and you correctly identify it, you earn {{ v_tp }} point. If you miss it, you lose {{ v_fn }} points.

  • If an intervention was not needed and you correctly identify that, you earn {{ v_tn }} point. If you mistakenly indicate that it was needed, you lose {{ v_fp }} point.

{% elif screen == 4 %}

Block 2 (Practice with DS)

What's Different: Now you'll see both the module reading AND a Decision Support (DS) system recommendation.

DS System: The DS is an AI system that learned from data and can help you by providing advice. The system shows a colored rectangle:

  • Red rectangle = "Alarm" (system recommends intervention)

  • Green rectangle = "No Alarm" (system says no intervention needed and you can skip)

Important: The DS system is not perfect - it can make mistakes. Be aware of its output and use your judgment to decide whether to follow its recommendation or not.

Remember: Make your own decision based on *all* available information. Scoring works the same as Block 1 - you'll get feedback after each decision.

{% elif screen == 6 %}

Block 3 (Main Experiment)

What's Different: This is the main experiment block. Everything works the same as Block 2:

  • You'll see the module reading

  • You'll see the DS system recommendation (red/green rectangle)

  • You decide: intervene or skip

  • You get feedback and points after each decision

Your score resets to 30 points for this block. The scoring rules remain the same.
